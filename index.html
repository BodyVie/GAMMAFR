Guide

GAMMA.FR, en une phrase

GAMMA.FR est le site qui permet aux joueurs francophones de télécharger la traduction française du modpack S.T.A.L.K.E.R. G.A.M.M.A., et de suivre son évolution.

Vue d'ensemble

Le site tient sur quelques onglets :

  • Files — le configurateur d'installation. Vous choisissez ce que vous voulez en français, le site fabrique une archive .zip prête à installer. C'est le cœur du site.
  • Liste — la liste de tous les mods traduits, avec une recherche pour vérifier si votre mod est couvert.
  • Changelog — l'historique des versions de la traduction.
  • Panneau d'affichage (accueil) — les annonces et nouveautés du moment, plus des liens utiles (ModDB, GitHub, Discord de G.A.M.M.A.).
  • Contact — pour signaler une erreur de traduction ou suggérer un mod à traduire.

Deux onglets se consultent simplement en lecture : Planner (la feuille de route) et Updates (les mods récemment mis à jour sur ModDB).

Notions clés

Seulement le vocabulaire propre au site :

  • Les 3 niveaux de traduction (choisis à la génération) :
    • base — uniquement G.A.M.M.A. de base.
    • tweak — base + les mods désactivés par défaut dans G.A.M.M.A.
    • extra — tout ça + des mods externes à G.A.M.M.A.
    • Chaque niveau supérieur inclut automatiquement le précédent.
  • Patch — un module de traduction pour un mod précis. Aux niveaux tweak/extra, vous cochez ceux que vous voulez.
  • Archive .zip — le fichier produit par le site, à installer dans Mod Organizer 2 (MO2) comme n'importe quel mod.

Mode opératoire

1 — Télécharger la traduction française

La tâche principale.

Pré-requis : aucun pour télécharger. Pour l'installer ensuite, avoir G.A.M.M.A. installé via Mod Organizer 2.

  1. Ouvrez l'onglet Files.
  2. Lisez le Lisez-moi en haut (rappel important : les fichiers sont modulables et peuvent entrer en conflit).
  3. À l'étape Niveau du configurateur, sélectionnez base, tweak ou extra.
  4. Cliquez sur Suivant ▸.
  5. (niveaux tweak et extra) À l'étape Patchs Tweak, cochez les patchs à inclure, puis cliquez sur Suivant ▸.
  6. (niveau extra) À l'étape Patchs Extra, cochez les mods externes à traduire, puis cliquez sur Suivant ▸.
  7. À l'étape Récapitulatif, vérifiez votre sélection.
  8. Cliquez sur Générer l'archive ▾.
  9. Patientez pendant le message « Téléchargement des fichiers… X / Y ».

Résultat attendu : une archive .zip (actuellement F.R.I.T.E.S.zip) se télécharge sur votre ordinateur.

Astuce : dans les étapes de patchs, utilisez le champ « Filtrer les patchs… » pour retrouver un mod vite. Le bouton ◂ Retour revient en arrière sans perdre votre sélection.

Vigilance : le niveau extra remplace davantage de fichiers. Si deux patchs cochés se chevauchent, le récapitulatif affiche un ⚠ — fiez-vous-y avant de générer.

Et ensuite ? (installation, hors site)

  1. Dans Mod Organizer 2, installez l'archive comme un mod (« Install Mod » ou glisser le .zip).
  2. Activez-la, puis placez-la tout en bas de votre liste (afin d'avoir la plus forte priorité).
  3. Désactiver : 308- Tactical Fonts for Anomaly 1k – CryoManne

N'activer pas 307- French Rework Improved Translation & Enhanced Style - Thundard

2 — Vérifier si un mod est déjà traduit

Pré-requis : connaître le nom du mod.

  1. Ouvrez l'onglet Liste.
  2. Saisissez le nom du mod dans le champ « Filtrer par nom… ».
  3. Lisez les résultats.

Résultat attendu : si le mod apparaît, il est couvert (niveau extra). S'il n'apparaît pas, il n'est pas encore traduit.

Astuce : s'il manque, suggérez-le via l'onglet Contact (voir tâche 3).

3 — Signaler une erreur ou suggérer un mod

Pré-requis : aucun.

  1. Ouvrez l'onglet Contact.
  2. (facultatif) Saisissez votre Pseudo.
  3. Dans Motif, choisissez Suggestion, Correction ou Autre.
  4. Saisissez un Objet (un résumé court).
  5. Rédigez votre Message.
  6. Cliquez sur Envoyer.

Résultat attendu : un message de confirmation s'affiche sous le formulaire.
